Annotation Tool

Инструкция:
1. Выбери класс (waveguide/flux/solder)
2. Обведи объект точками на фото (клик = точка)
3. Нажми "Замкнуть полигон" или клик рядом с первой точкой
4. Повтори для остальных классов
5. Нажми → для следующего кадра

Waveguide = стенки + полость (маленький прямоугольник)
Flux = белый порошок слева и справа
Solder = U-форма снизу
